# ImageReader

macOS App for demonstrate image analysis tool like Vision.framework.


## Feature

### Vision.framework
- [x] Text Recognization
- [ ] Object Detecting
- [ ] Face Detecting
- [ ] Feature Print
- [ ] Saliency

### CoreImage.framework
- [ ] Object Detecting
- [ ] Face Detecting

### OpenCV
- [ ] SURF Image Matching

### Tesseract
- [ ] Text Recognization


## Usage

This app is still under developing. Please build from source code via Xcode.app.
